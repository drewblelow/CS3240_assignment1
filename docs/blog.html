NUSMaids

NUSMaids started as a hackathon project, were we wanted create a service providing/fufilling web application. Our target user base was mainly NUS students, especially those residing on campus. We implemented basic features such as request posting, applying to provide services, as well as pricing and bargaining. Later on, the project idea was reused for a database application system. You can check it out over at NUSMaids.
